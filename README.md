## Library Management System

This is a web-based Library Management System developed  in 5th Semester in  Software Engineering subject.

##  Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP (Local Server)

## Features
- Admin login and dashboard
- Add, edit, delete books
- User registration and login
- Issue and view books
- User profile management
- Feedback system

## How to Run
1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project folder to `htdocs`.
4. Import the database in phpMyAdmin.
5. Open in browser:  
   `http://localhost/LMS project/`
